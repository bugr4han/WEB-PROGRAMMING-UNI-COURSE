<h2 align="center">PHP - Öğrenci Bilgi Hesaplama (Cookie)</h2>
<h4 align="center">Çerezleri (cookie) kullanarak kendisine GET metodu ile gönderilmiş bir sınıftaki öğrencilerin numarasını, vize ve final notlarını bir dizi halinde tutan, tüm listeyi görüntüledikten sonra en son satırda sınıfın genel ortalama notunu hesaplayarak görüntüleyen web dokümanının PHP kodunu yazınız.</h4>

---

```php
<form action="" method="get">
    Öğrenci Numarası: <input type="number" name="numara"><br>
    Vize Notu: <input type="number" name="vize"><br>
    Final Notu: <input type="number" name="final"><br>
    <input type="submit" value="Öğrenci Ekle">
</form>

<?php
    if(isset($_GET['numara'], $_GET['vize'], $_GET['final'])) {
        $numara = $_GET['numara'];
        $vize = $_GET['vize'];
        $final = $_GET['final'];
        $ogrenciler = $_COOKIE['ogrenciler'] ?? "";

        if($ogrenciler != "")
            $ogrenciler .= "\n";

        $ogrenciler .= "$numara,$vize,$final";
        setcookie('ogrenciler', $ogrenciler, time() + 3600);

        header("Location: ".$_SERVER['PHP_SELF']); // f5 atmadan yenilenmesi için
        exit;
    }

    $ogrenciler = $_COOKIE['ogrenciler'] ?? "";
    if($ogrenciler != "") {
        $ogrenciListesi = explode("\n", $ogrenciler);
        $toplamOrtalama = 0;
        $ogrenciSayisi = 0;

        echo "<table border='1'>";
        foreach($ogrenciListesi as $ogrenci) {
            list($numara, $vize, $final) = explode(",", $ogrenci);
            $ortalama = ($vize * 0.4) + ($final * 0.6);
            echo "  <tr>
                        <td>$numara</td>
                        <td>$vize</td>
                        <td>$final</td>
                        <td>" . $ortalama . "</td>
                    </tr>";
            $toplamOrtalama += $ortalama;
            $ogrenciSayisi++;
        }
        echo "</table>";

        $genelOrtalama = $toplamOrtalama / $ogrenciSayisi;
        echo "<h3>Genel Ortalama: " . $genelOrtalama . "</h3>";
    }
?>
```
