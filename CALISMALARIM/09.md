<h2 align="center">PHP - Matematiksel İşlemler</h2>
<h4 align="center">Matematiksel dört işlemin yapılabildiği iki adet web dokümanı oluşturunuz. Birinci doküman kullanıcıdan işlem türünü ve değerleri alacak, ikinci doküman ise alınan bilgilere göre işlemi gerçekleştirerek sonucu ekrana yazdıracaktır. Bu iki dokümana ait HTML ve PHP kodlarını yazınız.</h4>

---

```php
<form method="POST">
    <label for="number1">Num 1:</label>
    <input type="number" id="number1" name="number1"><br><br>
    <label for="number2">Num 2:</label>
    <input type="number" id="number2" name="number2"><br><br>

    <label for="islem">İşlem:</label>
    <select name="islem" id="islem">
        <option value="topla">Toplama işlemi</option>
        <option value="cikar">Çıkarma işlemi</option>
        <option value="carp">Çarpma işlemi</option>
        <option value="bol">Bölme işlemi</option>
    </select>
    <br><br>
    <input type="submit" value="Hesapla">
</form>

<?php
    if ($_SERVER["REQUEST_METHOD"] == "POST") {
        $num1 = $_POST["number1"];
        $num2 = $_POST["number2"];
        $islem = $_POST["islem"];
        $sonuc = "";

        switch ($islem) {
            case "topla":
                $sonuc = $num1 + $num2;
                break;
            case "cikar":
                $sonuc = $num1 - $num2;
                break;
            case "carp":
                $sonuc = $num1 * $num2;
                break;
            case "bol":
                if ($num2 != 0) {
                    $sonuc = $num1 / $num2;
                } else {
                    $sonuc = "Sıfıra bölme hatası!";
                }
                break;
        }

        echo "<h3>İşlem Sonucu:$sonuc</h3>";
    }
?>
```
