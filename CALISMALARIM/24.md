<h2 align="center">JS - Dinamik Resim Boyut Değiştirme</h2>
<h4 align="center">
Bir HTML formunda 100 adet resim bulunmaktadır ve bu resimlerin isimleri “resim1.jpg … resim100.jpg” şeklinde belirlenmiştir ve name/id olarak “resim1…resim100” şeklinde belirtilmiştir. Aynı şekilde, belirtilen resimlerin % oranında en ve boydan küçültülmüş küçük halleri mevcuttur ve “resimk1.jpg…resimk100.jpg” şeklinde isimlendirilmiştir. Her bir resme tıklanınca resim küçükse büyük büyükse küçük halini alacaktır. Belirtilen web dokümanının kodunu PHP/JavaScript/HTML kullanarak oluşturunuz.
</h4>

---

```html
<script>
    for (let i = 1; i <= 100; i++) {
        document.write(
            `<img src="resimk${i}.jpg" width="120" height="80" onclick="degistir(this)">`
        );
    }
    function degistir(resim) {
        const kW = 120, kH = 80, bW = 600, bH = 400;
        if (resim.width === kW) {
            resim.src = resim.src.replace("resimk", "resim");
            resim.width = bW;
            resim.height = bH;
        } else {
            resim.src = resim.src.replace("resim", "resimk");
            resim.width = kW;
            resim.height = kH;
        }
    }
</script>
```
