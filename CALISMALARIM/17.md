<h2 align="center">PHP - Sepet Uygulaması (Session)</h2>
<h4 align="center">Session yapısı kullanılarak iki sayfadan oluşan bir sepet uygulaması tasarlayınız. İlk sayfada ürün adı, ürün adedi ve fiyatı girilecektir. Her eklemeden sonra kullanıcı yine ilk sayfayı görecektir. Bu sayfadaki “Hesap” butonu kullanıcıyı ikinci sayfaya yönlendirecektir. İkinci sayfada tüm eklenen ürünler listelenecek ve en altta toplam tutar gösterilecektir. “Geri Dön” butonu ilk sayfaya döndürecek, “Tamamla” butonu ise sepeti temizleyip kullanıcıyı tekrar ilk sayfaya yönlendirecektir. İlgili PHP ve HTML kodlarını yazınız.</h4>

---

```php
index.php
------------
<form action="" method="POST">
    Ürün: <input type="text" name="urun" required>
    Adet: <input type="number" name="adet" required>
    Fiyat: <input type="number" name="fiyat" required><br><br>
    <input type="submit" value="Sepete Ekle" name ="ekle">
</form>
<form action="hesap.php"><input type="submit" value="Hesap"></form>

<?php
    session_start();
    if(!isset($_SESSION['sepet'])) $_SESSION['sepet'] = [];
    if(isset($_POST["ekle"])) {
        $_SESSION['sepet'][] = [$_POST["urun"], $_POST["adet"], $_POST["fiyat"]];
    }
?>
```

---

```php
hesap.php
-----------
<?php
    session_start();
    if (isset($_POST["tamamla"])) {
        session_destroy();
        header("Location:index.php");
        exit;
    }
    $toplam = 0;
    echo "<table border='1'>";
        echo "<tr>";
            echo "<th>Ürün</th>";
            echo "<th>Adet</th>";
            echo "<th>Fiyat</th>";
        echo "</tr>";
        foreach ($_SESSION['sepet'] as $s) {
            $toplam += $s[1] * $s[2];
            echo "<tr>";
                echo "<td>" . $s[0] . "</td>";
                echo "<td>" . $s[1] . "</td>";
                echo "<td>" . $s[2] . "</td>";
            echo "</tr>";
        }
    echo "</table>";
    echo "Toplam Tutar: " . $toplam;
?>

<form action="index.php" method="POST"> <input type="submit" value="Geri Dön"> </form>
<form action="" method="POST"> <input type="submit" value="Tamamla" name="tamamla"> </form>

```
