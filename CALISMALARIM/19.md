<h2 align="center">PHP - Link Listesi (Database)</h2>
<h4 align="center">192.168.1.1 adresinde çalışan, root kullanıcısının şifresi abc olan DENEME isimli veritabanında, LINKLER adlı ve (LinkID, hrefText, contextText) alanlarına sahip bir tablo bulunmaktadır. Bu tabloyu kullanarak, resim ve metnin birlikte gösterildiği linkleri alt alta listeleyen bir web sayfası oluşturunuz. Sayfaya ait PHP ve HTML kodlarını yazınız.</h4>

---

```php
<?php
    $con = new mysqli("192.168.1.1", "root", "abc", "DENEME");
    if ($con->connect_error) die("Hata");
    $res = $con->query("SELECT * FROM LINKLER");
    $rows = [];
    while ($row = $res->fetch_assoc())
        $rows[] = $row;
    $con->close();
    
    foreach ($rows as $row) {
	    printf(
	        '<div>
	            <a href="%s" target="_blank">
	                <img src="%s" width="100"><br>
	                %s
	            </a>
	        </div>', $row['hrefText'], $row['hrefText'], $row['contextText']);
    }
?>
```
