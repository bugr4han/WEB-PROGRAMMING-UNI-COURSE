<h2 align="center">PHP - Giriş Kontrolü (Cookie)</h2>
<h4 align="center">Kullanıcı adı ve şifre bilgilerinin girilebildiği, kendi kendini çağıran giris.php isimli bir PHP dosyası oluşturunuz. Bilgiler doğru girildiğinde “Giriş başarılı”, üç kez yanlış girildiğinde “Giriş reddedildi” mesajı üretilecektir. Giriş deneme sayısı her adımda ekranda gösterilecektir. Bu işlemi COOKIE yapısını kullanarak gerçekleştiren bir.php dosyasının kodunu yazınız.</h4>

---

```php
<form action="" method="POST">
    Nickname:<input type="text" name="nick" required>
    Password:<input type="password" name="pass" required>
    <input type="submit" value="Login" name="login">
</form>

<?php
    if (!isset($_COOKIE['deneme'])) setcookie('deneme', 0, time() + 3600);
    if (!isset($_COOKIE['giris'])) setcookie('giris', 0, time() + 3600);
    $deneme = $_COOKIE['deneme'] ?? 0;
    $giris = $_COOKIE['giris'] ?? 0;
    if ($_SERVER["REQUEST_METHOD"] == "POST") {
        $nick = $_POST['nick'];
        $pass = $_POST['pass'];
        if ($nick == "admin" && $pass == "1234") {
            $deneme = 0;
            setcookie('deneme', $deneme, time() + 3600);
            $giris = 1;
            setcookie('giris', $giris, time() + 3600);
            echo "Giriş Başarılı!";
            return;
        } else {
            $deneme++;
            setcookie('deneme', $deneme, time() + 3600);
            if ($deneme >= 3) {
                echo "Giriş Reddedildi!";
                return;
            } else {
                echo "Hatalı Giriş Tekrar Dene! - Giriş Denemesi:" . $deneme;
            }
        }
    }
?>
```
