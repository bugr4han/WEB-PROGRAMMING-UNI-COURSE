<h2 align="center">PHP - Giriş Kontrolü (Session)</h2>
<h4 align="center">Kullanıcı adı ve şifre bilgilerinin girilebildiği, session kullanılarak kendi kendini çağıran giris.php isimli bir PHP dosyası oluşturunuz. Kullanıcı adı ve şifre doğru girildiğinde “Giriş başarılı”, üç kez yanlış girildiğinde ise “Giriş reddedildi” mesajı gösterilecektir. Ayrıca giriş deneme sayısı session üzerinden tutulacak ve her aşamada ekranda görüntülenecektir. Bu işlemi gerçekleştiren giris.php dosyasının kodunu yazınız.</h4>

---

```php
<form action="" method="POST">
    Nickname:<input type="text" name="nick" required>
    Password:<input type="password" name="pass" required>
    <input type="submit" value="Login" name="login">
</form>

<?php
    session_start();
    if(!isset($_SESSION['deneme'])) $_SESSION['deneme'] = 0;
    if(!isset($_SESSION['giris'])) $_SESSION['giris'] = 0;
    if($_SERVER["REQUEST_METHOD"] == "POST") {
        $nick = $_POST["nick"];
        $pass = $_POST["pass"];
        if($nick == "admin" && $pass == "1234") {
            $_SESSION['deneme'] = 0;
            $_SESSION['giris'] = 1;
            echo "Giriş Başarılı!";
            return;
        } else {
            $_SESSION['deneme']++;
            if($_SESSION['deneme'] >= 3) {
                echo "Giriş Reddedildi!";
                return;
            } else {
                echo "Hatalı Giriş Tekrar Dene! - Giriş Denemesi:" . $_SESSION['deneme'];
            }
        }
    }
?>
```
