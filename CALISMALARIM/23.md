<h2 align="center">JS - Resim Galerisi Uygulaması</h2>
<h4 align="center">
İsimleri resim1 ile resim10 arasında olan 10 adet resmin bulunduğu kabul edilecektir. Sayfa, Buton – Resim – Buton şeklinde görünecektir. Sol butona basıldığında bir önceki, sağ butona basıldığında bir sonraki resim gösterilecektir. Yapı döngüsel çalışacak; ilk resimdeyken sola basıldığında son resim, son resimdeyken sağa basıldığında ilk resim görüntülenecektir. Bu yapıyı oluşturan HTML ve JavaScript kodlarını yazınız.
</h4>

---

```html
<input type="button" value="Prev" onclick="degistir(-1)" />
<img id="photo" width="120" height="80" />
<input type="button" value="Next" onclick="degistir(1)" />

<script>
  let index = 1;
  function degistir(adim) {
    index += adim;
    if (index < 1) index = 10;
    if (index > 10) index = 1;
    goster();
  }

  function goster() {
    document.querySelector("#photo").src = "resim" + index + ".jpg";
  }
  window.onload = goster;
</script>
```
