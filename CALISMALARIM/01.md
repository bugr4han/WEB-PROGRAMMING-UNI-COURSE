<h2 align="center">HTML - LINK </h2>

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8"/>
    <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
    <meta name="keywords" content="HTML LESSONS"/>
    <meta name="description" content="We Are Learning HTML"/>
    <title>LEARN HTML</title>
  </head>
  <body>
    <center>
      <!-- _blank: Yeni sekmede açar -->
      <a href="https://www.google.com" target="_blank">Google (blank)</a>

      <br/><br/>

      <!-- _self: Aynı sekmede açar (varsayılan) -->
      <a href="https://www.github.com" target="_self">GitHub (self)</a>

      <br/><br/>

      <!-- _parent: Üst çerçevede açar (frame kullanılıyorsa) -->
      <a href="https://www.twitter.com" target="_parent">Twitter (parent)</a>

      <br/><br/>

      <!-- _top: En üst pencere/çerçevede açar -->
      <a href="https://www.instagram.com" target="_top">Instagram (top)</a>
    </center>
  </body>
</html>
```
