<h2 align="center">JS - Matematiksel İşlemler</h2>

<h4 align="center">
İki sayısal değer için dört işlemin radio button’lar yardımıyla seçildiği bir web sayfası oluşturunuz. Hesaplama işlemi GET veya POST kullanılmadan, bir butonun tıklanmasıyla JavaScript üzerinden yapılacak ve sonuç aynı sayfada gösterilecektir. İlgili HTML ve JavaScript kodlarını yazınız.Aynı doküman üzerinde GET, POST olmaksızın sadece JavaScript ile gerçekleştireceğiniz örnekte, işlem sonucunu görüntülemek için bir “button” türünden input nesnesi kullanılacak; nesnenin tıklama olayında hesaplama yapılacaktır — submit butonu ile yapılmayacaktır.
</h4>

---

```html
<input type="number" id="s1">
<input type="number" id="s2">
<input type="radio" name="islem" value="topla"> Toplama
<input type="radio" name="islem" value="cikar"> Çıkarma
<input type="radio" name="islem" value="carp"> Çarpma
<input type="radio" name="islem" value="bol"> Bölme
<input type="button" value="Hesapla" onclick="hesapla()">
<p id="sonuc"></p>

<script>
    function hesapla() {
        const s1 = Number(document.querySelector("#s1").value);
        const s2 = Number(document.querySelector("#s2").value);
        const islem = document.querySelector("input[name=islem]:checked").value;
        let sonuc = 0;
        switch (islem) {
            case "topla":
                sonuc = s1 + s2;
                break;
            case "cikar":
                sonuc = s1 - s2;
                break;
            case "carp":
                sonuc = s1 * s2;
                break;
            case "bol":
                if (s2 === 0) return;
                sonuc = s1 / s2;
                break;
        }
        document.querySelector("#sonuc").textContent = `Sonuç:${sonuc}`;
    }
</script>
```
