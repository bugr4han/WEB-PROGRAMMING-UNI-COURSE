<h2 align="center">JS - Checkbox Durum Değiştirme</h2>
<h4 align="center">
Bir web sayfasındaki checkbox türündeki bir input elemanının her tıklamada kendi durumunu tersine çeviren bir JavaScript fonksiyonu yazınız. Bu fonksiyonu kullanan örnek bir HTML input ifadesi ile birlikte yazınız.
</h4>

---

```html
<input type="checkbox" checked /> 
<input type="checkbox" />
<div id="container"></div>
<input type="button" value="Çevir" onclick="tumunuTersCevir()" />

<script>
  function tumunuTersCevir() {
    let cbs = document.querySelectorAll("input[type='checkbox']");
    cbs.forEach((cb) => {
      cb.checked = !cb.checked;
    });
  }

  function cbCreate() {
    const container = document.querySelector("#container");
    for (let i = 0; i < 6; i++) {
      const cb = document.createElement("input");
      cb.type = "checkbox";
      if (i % 2 === 0) cb.checked = true;
      container.appendChild(cb);
    }
  }
  window.onload = cbCreate;
</script>
```
