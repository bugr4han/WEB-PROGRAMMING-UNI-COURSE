<h2 align="center">PHP - Asal ve Armstrong Sayı Kontrolü</h2>
<h4 align="center">Bir sayı girilebilen ve yapılacak işlemin radio button kullanılarak seçildiği bir PHP uygulaması oluşturunuz. Kullanıcı, radio button aracılığıyla asal sayı kontrolü veya Armstrong sayı kontrolü seçeneklerinden birini seçecektir. Seçilen işleme göre girilen sayının asal mı yoksa Armstrong sayı mı olduğu kontrol edilecek ve sonuç ekranda görüntülenecektir.</h4>

---

```php
<form action="" method="post">
    Sayı: <input type="number" name="sayi"><br><br>
    <input type="radio" name="islem" value="asal"> Asal sayı kontrolü<br>
    <input type="radio" name="islem" value="armstrong"> Armstrong sayı kontrolü<br><br>
    <input type="submit" value="Kontrol Et">
</form>

<?php
    if($_SERVER["REQUEST_METHOD"] == "POST") {
        $sayi = $_POST["sayi"] ?? 0;
        $islem = $_POST["islem"] ?? "asal";

        if($islem == "asal") {
            $asalMi = true;

            if($sayi <= 1) {
                $asalMi = false;
            } else {
                for($i = 2; $i < $sayi; $i++) {
                    if($sayi % $i == 0) {
                        $asalMi = false;
                        break;
                    }
                }
            }
            echo $asalMi ? "<p>$sayi bir asal sayıdır.</p>" : "<p>$sayi bir asal sayı değildir.</p>";
        } else if($islem == "armstrong") {
            $toplam = 0;
            $temp = $sayi;
            $basamak = strlen((string)abs($sayi));

            while($temp > 0) {
                $rakam = $temp % 10;
                $toplam += pow($rakam, $basamak);
                $temp = floor($temp / 10);
            }

            echo ($toplam == $sayi) ? "<p>$sayi bir Armstrong sayısıdır.</p>" : "<p>$sayi değildir.</p>";
        }
    }
?>
```
