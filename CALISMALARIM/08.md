<h2 align="center">PHP - Kartezyen Çarpım</h2>
<h4 align="center">Kullanıcıdan satır başlangıcı, satır bitişi, sütun başlangıcı ve sütun bitişi değerleri alınacaktır. Program, bu değerleri kullanarak satır aralığındaki her sayı ile sütun aralığındaki her sayının Kartezyen çarpımını oluşturacak ve bu eşleşmeleri tablo biçiminde ekranda görüntüleyecektir.</h4>

---

```php
<form method="post">
    Satır Başlangıcı: <input type="number" name="satir_bas" required><br>
    Satır Bitişi: <input type="number" name="satir_bit" required><br>
    Sütun Başlangıcı: <input type="number" name="sutun_bas" required><br>
    Sütun Bitişi: <input type="number" name="sutun_bit" required><br>
    <input type="submit" value="Hesapla">
</form>
<?php
    if ($_SERVER["REQUEST_METHOD"] == "POST") {
        $satir_bas = (int)$_POST["satir_bas"];
        $satir_bit = (int)$_POST["satir_bit"];
        $sutun_bas = (int)$_POST["sutun_bas"];
        $sutun_bit = (int)$_POST["sutun_bit"];

        echo "<h3>Kartezyen Çarpım Tablosu:</h3>";

        echo "<table border='1'>";
            for ($i = $satir_bas; $i <= $satir_bit; $i++) {
                echo "<tr>";
                    for ($j = $sutun_bas; $j <= $sutun_bit; $j++) {
                        echo "<td>$i x$j</td>";
                    }
                echo "</tr>";
            }
        echo "</table>";
    }
?>
```
