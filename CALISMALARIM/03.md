<h2 align="center">HTML - LIST</h2>

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8"/>
    <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
    <meta name="keywords" content="HTML LESSONS"/>
    <meta name="description" content="We Are Learning HTML"/>
    <title>LEARN HTML</title>
  </head>
  <body>
    <!-- Unordered list (Sırasız liste) -->
    <!-- type: disc (varsayılan), circle, square -->
    <ul type="disc">
      <li>Coffee</li>
      <ul type="circle">
        <li>Black Coffe</li>
        <li>Latte</li>
      </ul>
      <li>Tea</li>
      <li>Milk</li>
    </ul>

    <!-- Ordered list (Sıralı liste) -->
    <!-- type: 1, A, a, I, i -->
    <!-- start: kaçtan başlayacağını belirler -->
    <!-- reversed: sıralamayı ters çevirir (büyükten küçüğe) -->
    <ol type="A" start="3" reversed>
      <li>Coffee</li>
      <li>Tea</li>
      <li>Milk</li>
    </ol>

    <!-- Description list (Tanım listesi) -->
    <dl>
      <dt>Coffee</dt>
      <dd>- black hot drink</dd>

      <dt>Milk</dt>
      <dd>- white cold drink</dd>
    </dl>
  </body>
</html>
```
