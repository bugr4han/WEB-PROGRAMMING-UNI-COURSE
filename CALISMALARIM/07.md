<h2 align="center">Dizi İşleme ve Çıktı Oluşturma</h2>
<h4 align="center">Aşağıda verilen çıktı örneğinde, verinin POST metodu ile dizi şeklinde gönderildiğini kabul ediniz. foreach döngüsü veya uygun döngüler kullanarak aynı çıktıyı üreten PHP kodunu yazınız.</h4>

---

```java
OUTPUT:
Ankara
- Çankaya
- Etimesgut
- Gölbaşı
Amasya
- Merzifon
- Taşova
Ağrı
- Doğubayazıt
- Patnos
Afyon
- Çay
- Bolvadin
- Dinar
```

---

```php
<?php
    $sehirler =
    [
        "Ankara" => ["Çankaya", "Etimesgut", "Gölbaşı"],
        "Amasya" => ["Merzifon", "Taşova"],
        "Ağrı" => ["Doğubayazıt", "Patnos"],
        "Afyon" => ["Çay", "Bolvadin", "Dinar"]
    ];
?>
<ol>
    <?php
        foreach ($sehirler as $sehir => $ilceler) {
            echo "<li>";
                echo $sehir;
                echo "<ul>";
                    foreach ($ilceler as $ilce) {
                        echo "<li>" . $ilce . "</li>";
                    }
                echo "</ul>";
            echo "</li>";
        }
    ?>
</ol>
```
