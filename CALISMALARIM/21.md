<h2 align="center">PHP - Sorgu Çalıştırıcı (Database)</h2>
<h4 align="center">
GET metodu ile gönderilen bir SQL ifadesini kullanarak, 192.168.1.100 adresinde çalışan ve root kullanıcısının şifresi 123 olan DENEME veritabanında sorgu çalıştıran bir PHP sayfası oluşturunuz. Sorgu sonucu, sütun isimleriyle birlikte HTML tablo formatında ekranda gösterilecektir. İlgili PHP ve HTML kodlarını yazınız.
</h4>

---

```php
<form method="GET">
    SQL: <input type="text" name="sql">
    <input type="submit" value="Çalıştır">
</form>

<?php
    $con = new mysqli("192.168.1.100", "root", "123", "DENEME");
    if ($con->connect_error) die("Hata");
    $res = $con->query($_GET["sql"]);
    $rows = []; $fields = [];
    while ($field = $res->fetch_field())
        $fields[] = $field->name;
    while ($row = $res->fetch_assoc()) {
        $rows[] = $row;
    }
    $con->close();

    echo "<table border='1'>";
        echo "<tr>";
        foreach ($fields as $field)
            echo "<th>$field</th>";
        echo "</tr>";
        foreach ($rows as $row) {
            echo "<tr>";
            foreach ($fields as $field)
                echo "<td>" . $row[$field] . "</td>";
            echo "</tr>";
        }
    echo "</table>";
?>
```
